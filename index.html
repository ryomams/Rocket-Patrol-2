  • ← mode gives you only 4 seconds to play the game and makes the ships faster. → mode gives you a whole minute
  • When the game ends, you can press R to restart in the same difficulty or press the left arrow to return to the difficulty selection.
  • Press F to fire, use arrow keys to move
  • Press SPACE to toggle control modes
    • Mode 1 - Use the left and right arrow keys to move the ship. Mode 2 - Use A and D to move the ship. Mode 3 - Use the mouse to move the ship. Mode 4 - Disable movement
  • Press J to switch weapons
    • Weapon 1 - Fires faster
    • Weapon 2 - Fires slower
  • Destroying ships will give you ((point value of the ship / 4) * 1000) miliseconds back of time.
    • You can't get negative seconds, you're capped at 0 seconds sorry
  • Reload the page to return to the main menu.
